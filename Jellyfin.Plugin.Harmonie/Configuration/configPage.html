Harmonie connection

Base URL of the harmonie service. No trailing slash. Example: http://harmonie:8842.
Sent as X-API-Key. Leave empty if harmonie was started without authentication.

Path mappings

Optional. Used as a last-resort fallback if a track can't be matched by tags. Translate the harmonie-side path prefix to the Jellyfin-side prefix. One per line, format: /harmonie/path => /jellyfin/path.

Smart playlists

The plugin watches for two kinds of playlist names. Add tracks to one and the rest fills in automatically.
  • [RADIO] — track radio. Add one or more tracks and harmonie returns 20 similar ones.
  • [DRIFT] — drifting walk. Add a single track and harmonie generates 30 tracks that gradually move away from it.
Append n=<count> to override length, e.g. [DRIFT n=50] Long mix.
