[RADIO] Defaults

The first N tracks of a [RADIO] playlist are the seeds. Anything below them gets replaced on each refresh. Reorder a track into the top N to make it a seed. 1–20.
Number of tracks. 1–500.
0 is deterministic; 1 uses the most variation harmonie allows while preserving similarity.
Maximum BPM gap between consecutive tracks. Leave empty for no constraint.
When on, consecutive tracks must share a Camelot-compatible key (same key, ±1 wheel position, or parallel mode). Tracks without key info are excluded.

Number of tracks. 1–500.

How many tracks the playlist stays on each anchor before re-anchoring on the most recent pick. Larger values stay closer to the seed; smaller drifts faster. 1–100.
Maximum BPM gap between consecutive tracks. Leave empty for no constraint.

[MIX] Defaults

Number of tracks. 1–500.
0 is deterministic; 1 uses the most variation harmonie allows while preserving similarity.
How far back to look for seed tracks. 1–365.
Maximum number of seeds taken from your listening history. The plugin picks the top N most recently played tracks within the listening window, or the top N most played when "Seed by top-played" is on. 1–100.
Off: seeds are the most recently played tracks in the listening window. On: seeds are the highest play-count tracks in the window. Override per playlist with [MIX top].
Off: harmonie returns tracks similar to the seeds. On: harmonie walks away from the seeds for more variety. Override per playlist with [MIX drift].
Maximum BPM gap between consecutive tracks. Leave empty for no constraint.
When on, consecutive tracks must share a Camelot-compatible key (same key, ±1 wheel position, or parallel mode). Tracks without key info are excluded.

[STYLE] / [GENRE] Defaults

Number of tracks. 1–500.
Minimum classifier confidence a track needs to qualify. 0.0 lets every loosely-tagged track through; 1.0 only the strongest matches. Override per playlist with style_min=F. 0.0–1.0.

Instant Mix / Song Radio

Replaces Jellyfin's built-in Instant Mix (also called Song Radio in clients like Finamp) with audio-similarity matches from harmonie. When the seed track isn't in harmonie's index, harmonie returns nothing, or the service is unreachable, the plugin falls back to Jellyfin's default genre-based behaviour so the endpoint always returns something.
0 always returns the same ranking; 1 uses the most variation harmonie allows while preserving similarity.

Personal Mix playlists

Optional. The 30-day refresh task creates and maintains activity-derived playlists for each Jellyfin user. Adjust the cadence under Scheduled Tasks. As tastes shift, playlists rename and refill in place.
The actual count adapts to recent listening history. Reducing the maximum deletes excess playlists. 0–10.
How far back to look when computing each user's top styles. 1–365.
Number of tracks in each Personal Mix playlist. 1–500.
0 is deterministic; 1 uses the most variation harmonie allows while preserving similarity.

Path mappings

Optional. Use these only when harmonie and Jellyfin see your music library at different filesystem paths. The plugin uses them as a fallback when a track can't be matched by tags. Format: /harmonie/path => /jellyfin/path, one per line.
