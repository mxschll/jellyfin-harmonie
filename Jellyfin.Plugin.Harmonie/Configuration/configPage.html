Harmonie connection

Base URL of the harmonie service. No trailing slash. Example: http://harmonie:8842.
Sent as X-API-Key. Leave empty if harmonie was started without authentication.

Path mappings

Optional. Used as a last-resort fallback if a track can't be matched by tags. Translate the harmonie-side path prefix to the Jellyfin-side prefix. One per line, format: /harmonie/path => /jellyfin/path.

Smart playlists by name prefix

Any Jellyfin playlist whose name starts with the prefix below is auto-extended by harmonie. Three modes, chosen via the playlist title:
  • [HRMNY] — seed mode. Add 1+ tracks; harmonie returns similar ones.
  • [HRMNY drift] or [HRMNY drift=10] — drift mode. One seed; the playlist walks away from it. Optional chunk size.
  • [HRMNY energy=80] — energy mode. No seed needed; harmonie picks tracks at that energy level (0–100, mapped to danceability) and shuffles them.
Append n=<count> to set total playlist length, e.g. [HRMNY drift n=40].
